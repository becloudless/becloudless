Build and maintain your own infrastructure easily, from zero to enterprise-level, using only free and open-source software.

BeCloudLess provides the opinionated glue between awesome FOSS tools to make what might seem like a complex infrastructure a dead-simple stack to operate.

## Project Status

BeCloudLess is in early development. It is used in production by the author, but the architecture is not yet stable and breaking changes can occur at any time. Stable architecture and self-migration tooling is planned for V1.
