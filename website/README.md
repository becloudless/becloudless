# BeCloudLess Documentation Website

Hugo site using the [Relearn](https://mcshelby.github.io/hugo-theme-relearn/) theme.

## Setup

```bash
cd website
hugo mod get -u
```

## Development

```bash
hugo server
```

Open <http://localhost:1313/>.

## Build

```bash
hugo --minify
```

Output is written to `public/`.
