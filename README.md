# BeCloudLess

Build and maintain your own infrastructure easily, from zero to enterprise level infrastructure using only free and open-source software.


BeCloudLess only do, with an opinionated architecture, the glue between awesome free and open source software, to make what can be (seen as) a complex infrastructure, a dead simple stack to use.


# Key concept

BeCloudLess is meant to ease your installation of physical devices (laptop, desktops, servers, TVs, ...), till running a full-fledged infrastructure,
with all needed services to be independent of well known companies, like google, amazon, microsoft, apple, facebook, etc.,

Yes, open source is so powerful that you can be your own Google.
